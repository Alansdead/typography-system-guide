# Typography System Guide for iOS and Android

A comprehensive interactive guide for implementing accessible typography in mobile applications, covering both iOS and Android platforms.

## What’s New

See [CHANGELOG.md](CHANGELOG.md) for the latest updates and release history.

## Features

- Complete breakdown of iOS text styles with detailed specifications — including all 12 Dynamic Type sizes (xSmall → AX5)
- Material Design 3 typography styles for Android with Android 14+ non-linear scaling documentation
- Interactive demonstration of Dynamic Type (iOS) and Font Scaling (Android, up to 2.0x)
- Accessibility guidelines and best practices (WCAG 2.2 AA compliant, covering SC 1.4.4, 1.4.10, 1.4.12, and the new SC 2.5.8 Target Size)
- Implementation code samples for UIKit, SwiftUI, Android XML, and Jetpack Compose
